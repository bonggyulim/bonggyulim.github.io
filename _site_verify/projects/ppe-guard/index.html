Overview

작업자 보호구 착용 여부를 검사하고 결과를 저장·조회할 수 있도록 구성한 사내망 배포형 AI 검사 시스템입니다.

Role

백엔드 구조 설계, AI 판정 연동, 고객사 DB 연계 방식 설계, Docker 실행 환경 구성, 환경 변수 기반 운영 구조 정리를 담당했습니다.

Tech Stack

  • Flask
  • Python
  • Docker
  • Customer DB
  • OpenCV

Key Features

  • 웹캠 및 영상 입력 기반 검사
  • AI 판정 결과 저장 및 조회
  • 백엔드를 통한 DB 접근 제어
  • .env 기반 환경 설정
  • 사내망 배포 운영

Architecture

프론트엔드 요청을 Flask API가 수신하고, AI 판정과 결과 저장을 백엔드에서 통제하는 구조로 구성했습니다.
