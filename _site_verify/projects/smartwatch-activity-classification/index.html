Overview

UCI WISDM 스마트워치 센서 데이터를 활용해 Walking, Sitting, Standing 활동을 분류한 머신러닝 프로젝트입니다. 가속도계와 자이로스코프 시계열을 전처리하고, sliding window 기반 샘플 생성과 특징 추출을 통해 분류 모델을 구성했습니다.

Role

센서 데이터 전처리, 특징 엔지니어링, 분류 실험 설계, 2-stage 분류 전략 검토, Streamlit 기반 시각화 프로토타입 구현을 담당했습니다.

Tech Stack

  • Python
  • Pandas
  • Scikit-learn
  • Streamlit

Key Features

  • UCI WISDM 스마트워치 센서 데이터 활용
  • accel / gyro 기반 x, y, z 시계열 데이터 전처리
  • sliding window 기반 샘플 생성
  • magnitude 및 통계 특징 추출
  • Random Forest baseline 실험
  • Walking vs Static, Sitting vs Standing 2-stage 분류 실험
  • Streamlit 기반 예측 및 시각화 프로토타입 구현

Architecture

센서 원시 데이터를 window 단위 샘플로 변환하고, 특징 추출 후 분류 모델에 연결하는 전형적인 시계열 머신러닝 파이프라인으로 구성했습니다.

Result

센서 기반 활동 분류 문제를 baseline과 2-stage 접근으로 비교할 수 있는 구조를 만들었고, 예측 결과를 Streamlit으로 빠르게 확인할 수 있는 프로토타입까지 정리했습니다.

What I Learned

시계열 센서 데이터에서는 모델 복잡도보다도 window 설계, 특징 추출, 클래스 분리 전략이 결과에 큰 영향을 준다는 점을 확인했습니다.
