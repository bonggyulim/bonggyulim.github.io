Developer Profile

저는 AI 모델을 단순 실험으로 끝내지 않고, 실제 서비스 흐름 안에서 동작하도록 연결하는 과정에 관심이 많은 개발자입니다.

요구사항 정의, 데이터 흐름 설계, API 구조, DB 스키마, 파일 저장소, 배포 환경을 함께 고려하며 프로젝트를 설계하고 구현했습니다.

특히 산업 AI 이상탐지 플랫폼과 PPE 검사 시스템에서는 기획, 설계, 주요 기능 구현, Docker 기반 실행 환경 구성, 인프라 구축까지 폭넓게 담당했습니다.
