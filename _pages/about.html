개발자 소개

저는 AI 모델을 단순 실험으로 끝내지 않고, 실제 서비스 흐름 안에서 동작하도록 연결하는 과정에 관심이 많은 개발자입니다.

요구사항 정의, 데이터 흐름 설계, API 구조, DB 스키마, 파일 저장소, 배포 환경을 함께 고려하며 프로젝트를 구현해왔습니다.

산업 AI 이상탐지 플랫폼과 PPE Guard 프로젝트에서는 기획, 설계, 주요 기능 구현, Docker 기반 실행 환경 구성까지 폭넓게 담당했습니다.

관심 영역

AI 서비스 백엔드 FastAPI Spring Boot RAG / LLM 이미지 기반 이상탐지 Docker 기반 배포 기술 문서화

기술 스택

Backend

Spring Boot, FastAPI, Flask

API 설계, 서비스 로직 구현, 운영 구조 설계

AI / ML

PyTorch, DINOv2, PatchCore, EfficientAD, RAG / LLM

실험 구조 설계, 추론 파이프라인 구성

Database / Storage

MariaDB, Redis, ChromaDB, MinIO

데이터 저장, 검색, 캐시, 파일 관리

Infra

Docker, Docker Compose, Nginx, GitHub Actions

배포 환경 구성, 실행 자동화, 운영 지원
